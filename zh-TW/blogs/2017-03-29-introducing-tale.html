泰勒的簡介

Tale是一個最小的Jekyll主題，為故事講述者策劃。 它由我為寫短篇小說的朋友設計和開發。

泰勒的特點

  • 與 GitHub 頁面兼容
  • 響應式設計(在行動裝置上看起來非常棒)
  • 語法高亮顯示，借助 Pygments
  • Markdown 和 HTML 文本格式
  • 文章的分頁

前往示例內容發布展示泰勒的文本格式功能。

跨瀏覽器支持

故事適用於大部分(如果不是全部)現代瀏覽器，包括Chrome，Safari和Firefox👍🏼

下載或貢獻

泰勒在 GitHub 上開放源碼，因此請繼續下載或在GitHub上複製它。 如果您發現任何錯誤或有任何建議，請隨時創建問題或提出請求。
